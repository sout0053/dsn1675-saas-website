Tour

Let's take a look at the features.

Ability to sign up for an account for free

Ability for paid members to create project, delete project, and edit project

Ability to buy membership, buy individual animation, or download animation free in low res

Ability for user to upload logo/image

Ability for user to select which animation they want to apply to image

Ability for user to manipulate different aspects of the selected animation using a 0-100% slider bar. Ex-if a bounce animation is chosen, aspects of that could be bounce speed, height of bounce, squish/no squish when bouncing, etc
